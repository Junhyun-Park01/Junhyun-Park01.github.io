# Junhyun-Park01.github.io

Personal academic homepage of **Junhyun Park** — a researcher working on
surgical robotics and medical AI.

🔗 **Live site:** https://junhyun-park01.github.io/

## About

A lightweight static site (plain HTML/CSS, no build step) hosted on GitHub Pages.
It lists publications, experience, education, awards, and a photos page.

## Structure

| Path | Description |
|------|-------------|
| `index.html` | Main page — profile, news, publications, experience, education, awards |
| `photos.html` | Photo gallery |
| `stylesheet.css` | Site styles (CV-aligned typography) |
| `publication_images/` | Publication thumbnails and profile photo |
| `life_images/` | Photos page images |
| `data/` | CV PDF and other assets |

## Local preview

```bash
python3 -m http.server 8000
# open http://127.0.0.1:8000/index.html
```
